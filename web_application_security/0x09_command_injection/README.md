# 0x09. Web Application Security - Command Injection

## Description
This project explores Command Injection vulnerabilities in web applications. It demonstrates how improper input validation can allow an attacker to execute arbitrary OS commands on the host operating system.

## Topics Covered:
* Command Injection fundamentals
* Bypassing input filters
* Command chaining (e.g., using `;`, `&&`, `|`)
* Reading sensitive files (e.g., `/etc/passwd`, flags)
