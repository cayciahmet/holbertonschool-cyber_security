# 0x08. Server-Side Request Forgery (SSRF)

## Description
This project focuses on identifying and exploiting SSRF vulnerabilities in web applications. By manipulating server-side requests, we gain access to internal resources that are not publicly accessible.

## Tasks
* **0. Unlocking security, one exploit at a time!**: Exploited an SSRF vulnerability in the `articleApi` parameter to access an internal admin dashboard on port 3000 and retrieve a hidden flag.
