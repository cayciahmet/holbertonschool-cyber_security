# 0x07. Web Application Security - File Inclusion

## Description
This project focuses on understanding, identifying, and exploiting File Inclusion and Template Injection vulnerabilities in web applications.

## Topics Covered:
* Local File Inclusion (LFI)
* Remote File Inclusion (RFI)
* Directory / Path Traversal
* Server-Side Template Injection (SSTI)
* Environment Variable Extraction
